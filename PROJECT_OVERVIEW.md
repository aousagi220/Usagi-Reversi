# USAGI REVERSI - プロジェクト概要

> ChatGPT向けのプロジェクト説明ドキュメントです。

---

## プロジェクト概要

**名称:** USAGI REVERSI（うさぎリバーシ）
**種別:** Webブラウザで動作するリバーシゲーム
**技術スタック:** HTML / CSS / Vanilla JavaScript

---

## 今ある機能

### ゲームロジック

| 機能 | 詳細 |
|------|------|
| ボード管理 | 8×8 のマスを2次元配列で管理。ゲーム開始時に中央4マスへ初期石を配置 |
| 石の配置検証 | クリックしたマスに置けるか 8方向（上下左右＋斜め4方向）をチェック |
| 石の反転 | 挟んだ相手の石を即時に反転。1ターンで複数方向の反転にも対応 |
| ターン管理 | 黒→白→黒... と自動で手番を切り替え |
| パス処理 | 置ける場所がないプレイヤーは自動でパス（相手のターンへ） |
| ゲーム終了判定 | 両プレイヤーが連続でパスになったらゲーム終了と判定 |
| 勝敗判定 | 終了時に石の数を比較し「黒の勝ち / 白の勝ち / 引き分け」を判定 |
| スコア集計 | 盤面の黒石・白石の総数をリアルタイムで集計 |
| CPU対戦 | 人間 / CPU を黒・白それぞれで切り替え可能。「弱いCPU」「普通CPU」「強いCPU」を実装済み |
| 弱いCPU | 置けるマスの中からランダムに1手を選ぶ |
| 普通CPU | 置けるマスの中から、最も多く石を取れる1手を選ぶ。同じ枚数を取れる手が複数ある場合はランダムに選ぶ |
| 強いCPU | 反転枚数、角、辺、角周辺の危険マス、序盤の1枚返しを評価して最も高得点の手を選ぶ |

### UI / 表示

| 機能 | 詳細 |
|------|------|
| ボード描画 | CSS Grid で 8×8 のセルを動的生成 |
| 有効手ハイライト | 現在の手番で置けるマスを緑色で表示 |
| スコアパネル | 黒・白それぞれの石数と現在の手番をリアルタイム表示 |
| 勝利モーダル | ゲーム終了時に勝者・最終スコア・再戦ボタンをモーダルで表示。出現時にアニメーションあり |
| リセット / 再戦ボタン | ゲームを初期状態に戻す（サイドパネルのリセットボタンと勝利モーダルの再戦ボタンの2箇所） |
| パス通知トースト | パス発生時に「黒はパスされました！」「白はパスされました！」を一時表示 |
| CPU設定UI | 黒・白それぞれでCPUのON/OFFと難易度を選択可能 |
| レスポンシブ対応 | デスクトップ・タブレット・モバイルの3段階で画面サイズに対応 |

## 次に実装する機能

### CPU自動対戦とテスト

- CPU同士を複数回自動対戦させ、先手・後手別の勝率を記録する
- 石の反転、パス、ゲーム終了、CPUの手選択を自動テストする
- 対戦結果をもとに強いCPUの評価点を調整する

---

## 将来的に実装してみたい機能

### CPU / AI の強化

**検討中のアイデア**

| レベル | 状態 | アルゴリズム案 |
|--------|------|----------------|
| 弱い CPU | 実装済み | ランダムに置ける場所を選ぶ |
| 普通の CPU | 実装済み | 最も多く石を取れるマスを選ぶ（貪欲法） |
| 強い CPU | 実装済み | 角を優先・危険マスを避けるヒューリスティック |
| AI | 未実装 | ミニマックス法 / アルファベータ枝刈りなど |

**課題・検討点**

- CPU自動対戦による勝率の計測
- 盤面を仮置きし、相手の合法手数を評価する処理の追加
- ミニマックス法やアルファベータ枝刈りによる先読み
- 将来的に機械学習（強化学習）を使ったAIも検討したい

## ファイル構成（参考）

```
Reversi/
├── index.html               # ゲーム画面のHTML
├── index.css                # スタイル・レスポンシブ対応
└── JS/
    ├── script.js            # 定数・グローバル変数の定義、クリック／リセット／再戦イベントの登録
    ├── UI.js                # DOM描画・表示更新（ボード描画・スコアパネル・モーダル制御）
    ├── functionBoard.js     # ボード初期化・初期石配置
    ├── functionStone.js     # 石の配置検証・反転・スコア集計・ゲーム終了判定
    ├── functionGameFlow.js  # ゲーム進行制御（ターン切り替え・パス処理・ゲーム終了・リセット）
    └── functionCpuFlow.js   # CPUの手番制御・CPUの手選択ロジック
```
