# Usagi Reversi

青うさぎが勉強用に作成した、JavaScriptで動くリバーシ（オセロ）です。
ブラウザだけで遊べます。

## 実行方法

1. 右上の `<> Code` より、`Download ZIP` を行ってください。
2. ZIPファイルを解凍してください。
3. `index.html` を Chrome などのブラウザで開いてください。

または、以下のURLから遊べます。

https://aousagi220.github.io/Usagi-Reversi/

- テスト方法と学習方法: [docs/TEST_AND_TRAINING.md](docs/TEST_AND_TRAINING.md)

## 機能

- 石の配置・反転・ターン管理などリバーシの基本ルールに対応
- 置けるマスの緑色ハイライト表示
- 黒・白の石数をリアルタイムでスコアパネルに表示
- 置ける場所がない場合は自動でパス
- パス発生時に画面上へ通知を表示
- ゲーム終了時に勝敗をモーダルで表示
- 黒・白それぞれで人間 / CPU を切り替え可能
- 弱・普通・強・学習AIの4段階から対戦相手を選択可能
- デスクトップ・タブレット・モバイルに対応したレスポンシブUI

## CPUについて

- 弱いCPU: 実装済み。置けるマスからランダムに選びます。
- 普通CPU: 実装済み。置けるマスの中から最も多く石を取れる手を選びます。
- 強いCPU: 角・辺を優先し、空いている角の隣のCマス・Xマスを避けます。
- 学習AI: 学習済み評価モデル、置換表、時間制限付きαβ探索を使用します。
  既定350msで反復深化し、残り8マス以下では完全読みを試みます。

Node.jsの`Automation`にも弱・普通・強CPUがあり、対局シミュレーション、
定石生成、学習モデルの評価相手として使用します。詳しくは
[テストとAI学習](docs/TEST_AND_TRAINING.md)を参照してください。
